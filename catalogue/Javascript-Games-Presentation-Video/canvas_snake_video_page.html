Canvas Snake Video

Description:

Classic Snake Game developed using HTML Canvas for rendering.

The purpose of the game is to control the snake within the canvas, grabbing the fruit will raise the score and allow the snake to grow longer hence rewarding the player with points.

This version of the game does not allow the player to drive through walls so if the player collides with the tail or with the wall the game will end.
