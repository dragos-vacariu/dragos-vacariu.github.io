Snake Video

Description:

Classic Snake Game

The purpose of the game is to control the snake within the table, grabbing the fruit will raise the score and allow the snake to grow longer hence rewarding the player with points.

This version of the game allows the player to drive through walls so the only available collision is between the snake head and the tail. If the head hit the tail the game will end.
