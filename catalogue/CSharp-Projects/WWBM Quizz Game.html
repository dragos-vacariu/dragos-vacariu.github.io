WWBM Quizz Game

WWBM Quizz Game is a .NET GUI Game-like applications inspired by the TV Show: Who wants to be a millionare

The player has to answer correctly to general knowledge questions of various difficulty from various fields such as: Cinema, Music, Geography, Biology, Chemistry, Astronomy, Literature, Internet, Technology, History, and so on.

There are about 200 questions available within the game, every question is presented with 4 possible answers. As the player progresses and earns money, items beoome available withing the Item Shop. Among the available items are:

  • - chances so that when player doesn't answer correctly the game will allow him to continue further with different question;
  • - reduce the number of wrong answers each question have;
  • - money multipliers;
