WWBM Quizz Game is a desktop GUI application developed using the .NET Framework during the early days of learning programming. Inspired by the popular TV Show: Who Wants to Be a Millionaire?, the game offers a quiz-style experience designed to test the player's general knowledge across a wide range of topics.

The player has to answer correctly to general knowledge questions of various difficulty from various fields such as:
  • Cinema
  • Music
  • Geography
  • Biology
  • Chemistry
  • Astronomy
  • Literature
  • Internet
  • Technology
  • History
  • and so on.
The game includes a pool of approximately 200 unique questions, each with four possible answers. As players progress and earn virtual money by answering correctly, they unlock items in the Item Shop, which can enhance gameplay. Available items include:
  • Second Chance – allows the player to continue the game even after a wrong answer
  • Answer Reduction – eliminates one or more incorrect options from the question
  • Money Multipliers – increase the virtual earnings for correct answers

This project was a great opportunity to explore event-driven programming, data handling, and user interface design in C#.
