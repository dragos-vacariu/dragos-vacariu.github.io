TowerBlocks Game

Description:

TowerBlocks is a game in which the player has to drop blocks one of top of the other in order to build the talled skyscraper possible. The player has to sync each drop so that the block will land on the platform.

There are a limited number of blocks to spare, if the player miss dropping a block on the platform than the spare blocks will decrease. Once all the spare blocks are exhausted the game will end.

There are several settings available in the Main Menu which can alter the gameplay; Wind Effects, Drop Assists, Crane Speed, Gravity.

A similar project was developed using JavaScript, see: JavaScript TowerBlocks
