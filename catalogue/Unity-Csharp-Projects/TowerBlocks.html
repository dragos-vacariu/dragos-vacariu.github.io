TowerBlocks is a game where the player must drop blocks on top of one another to build the tallest skyscraper possible. Timing is key â€” each block must be carefully dropped to land on the platform below.

The player has a limited number of spare blocks. If a block misses the platform, one is lost. When all spare blocks are used up, the game ends.

Several gameplay settings are available in the Main Menu, including:
  • Wind Effects
  • Drop Assists
  • Crane Speed
  • Gravity
