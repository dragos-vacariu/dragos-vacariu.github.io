2D Traffic Simulator was an early ambitious project developed from scratch using C++ and OpenGL. The goal was to simulate a dynamic traffic environment with features such as:
  • Predefined driving patterns for AI-controlled vehicles
  • Dynamic speed variations
  • Simulated time progression with day-to-night transitions
  • Adaptive traffic density based on the in-game time (reduction and repopulation)

Although the project was never fully completed, it played a significant role in my growth as a developer. It offered hands-on experience in areas such as:
  • Core programming and algorithm design
  • Texture filtering and photo editing for visual assets
  • Code optimization and performance tuning
  • Low-level hardware interaction and OpenGL API usage
