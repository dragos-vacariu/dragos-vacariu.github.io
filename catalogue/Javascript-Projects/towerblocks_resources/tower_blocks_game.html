TowerBlocks is a timing-based stacking game where the player must drop blocks one on top of the other to build the tallest skyscraper possible. Precision is keyâ€”each block must be carefully timed to land accurately on the one below it.

The player starts with a limited number of spare blocks. Each time a block is missed and doesn't land on the platform, one spare is lost. The game ends when all spare blocks are used up.

Controls:

Once the game is started - press Drop Block button or the SPACE key to drop the blocks.
