Knight Chess is a puzzle game in which the player has to to move the Knight piece across each square on the chessboard, following the unique movement pattern of the Knight. The player must visit each square only once, carefully planning each move.

The game is won when the player successfully visit every square on the chessboard. If the player runs out of valid moves before covering all the squares, the game ends.
