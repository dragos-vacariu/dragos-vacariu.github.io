This is a browser-based version of the classic Snake game, developed using JavaScript, HTML, and CSS.

The objective of the game is to control a snake within a grid-based play area. Each time the snake collects a fruit, the score increases and the snake grows longerâ€”making the game progressively more challenging.

In this version, the snake can pass through walls without penalty. The only game-ending condition is if the snake's head collides with its own tail, adding a strategic element as the snake grows in length.
