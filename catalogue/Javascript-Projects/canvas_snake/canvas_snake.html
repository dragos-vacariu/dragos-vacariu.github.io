Snake is a game in which the player has to guide the snake across the playfield to collect fruit. Each fruit increases your score and extends the length of the snake, increasing the challenge as the game progresses.

Unlike the previous version, this build introduces stricter rules—colliding with either the wall or the snake’s own tail ends the game, raising the stakes for careful control and strategic movement.

Instructions:
  • Controls: Use the specified keys to move the snake.
  • Goal: Eat the fruit to grow longer and earn points.
  • Game Over: The game ends if you collide with your own tail or hit the wall.
