Java MP3 Player is a desktop music player application developed during my early programming days, using Java, the NetBeans 8.2 IDE, and the Swing GUI toolkit. It builds upon an earlier MP3 Player I created in C#, offering enhanced functionality and a more refined user interface.

Key improvements include:
  • A more polished and responsive GUI using Java Swing components
  • Support for creating, saving, and loading playlists via .txt files
  • Playlist functionality with serialization and deserialization, allowing users to save and reload their music collections across sessions
  • A visual playlist viewer using a JList (listbox), enabling users to browse and manage their current tracklist more easily
